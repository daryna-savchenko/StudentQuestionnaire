# StudentQuestionnaire
that's a GUI form for student data.

it stores data of a student, that include:
- name;
- surname;
- age;
- group;
- email;
- year of study;
- hobbies;
- gender;
- and info.

buttons:
- "save"
   wenn clicking, entered data is saved into student_profile.txt;
- "delete"
  wenn clicking, clears all fields in a form and resets gender, year and hobbies.

  it also checks input:
  - name,surname, age and group should be filled;
  - age must be a positive number;
  - email should contain "@" sign.
